# HarmfulCommentDetector
ثبت نظر و  تشخیص ناهنجاری

##اجرای پروژه
بعد از ایجاد محیط مجازی و نصب کتابخونه ها:
دیتاست رو دانلود کنید،از دیتاست زیر استفاده کردیم:
[PHICAD](https://github.com/davardoust/PHICAD)

بعد از دانلود در پوشه main استخراج کنید باید حاوی ۲ فایل باشد، با نام های زیر و اینکه تغییرش ندین.
PHICAD-part1.csv
PHICAD-part2.csv

فایل data_load.py اجرا کنید که دیتاست ها رو ادغام و تمیز میکنه و به عنوان یه فایل جدید ذخیره میکنه که برای آموزش مدل از اون استفاده کنیم
خب الان فایل train.py رو اجرا کنید که مدل رو آموزش میده و خروجی رو داخل پوشه models ذخیره میکنه.
در آخر هم با اجرای app.py که داخل پوشه controllers هستش میتونید به خروجی دسترسی داشته باشید.
